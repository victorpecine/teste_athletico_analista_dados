Etapa 0

Filtro base — universo de análise

Filtro: jogadores sub-21 com minutos reais em campo
Por quê: A tabela tem 1.114 linhas, mas 165 jogadores com 0 minutos (convocados/lesionados). E 3 registros com idade = 0 (dados sem preenchimento). O universo real de análise é 216 jogadores sub-21 que efetivamente jogaram. Todo cálculo deve partir desse filtro — senão médias e rankings ficam distorcidos por quem não pisou em campo.
Condição base DAX
idade >= 15 && idade <= 21 && minutos_jogados > 0
Guarda esse filtro em uma medida base. Todos os outros cálculos herdam ele. Resultado esperado: ~216 jogadores válidos.
Grupo 1

Volume de participação — quem realmente jogou?

1a. Percentual de jogos como titular
qt_jogos_titularqt_jogos
Por quê: Entrar só como substituto nos últimos 5 minutos infla o contador de jogos sem representar confiança do treinador. A relação titular/total mostra quem tem espaço consolidado no time.
Medida DAX
% Titular = DIVIDE( SUM(fato_stats_jogadores_2025[qt_jogos_titular]), SUM(fato_stats_jogadores_2025[qt_jogos]) )
Jogadores com >60% de titularidade têm presença real. Abaixo de 30% são reservas. Combine com minutos totais para confirmar.
1b. Minutos por jogo disputado
minutos_jogadosqt_jogos
Por quê: Um jogador com 10 jogos e 900 minutos (90 min/jogo) é muito mais relevante que outro com 10 jogos e 200 minutos (20 min/jogo). Esta métrica mostra intensidade de uso.
Medida DAX
Min por Jogo = DIVIDE( SUM(fato_stats_jogadores_2025[minutos_jogados]), SUM(fato_stats_jogadores_2025[qt_jogos]) )
Referência: 90 min = jogo inteiro. Acima de 60 min/jogo indica jogador relevante no time. Use como critério de corte mínimo para o ranking final.
1c. Threshold de minutos mínimos para o ranking
minutos_jogadosminutos_jogados_divid_90
Por quê: Vitor Roque jogou 2.117 minutos (~23 jogos completos). Um jogador com 90 minutos totais não pode estar no mesmo ranking. O campo minutos_jogados_divid_90 já faz essa conversão — use-o para definir um piso. O critério mais usado em análise de scouts é mínimo de 5 jogos de 90 minutos (= 450 min).
Sugestão de filtro para ranking
minutos_jogados_divid_90 >= 5 -- equivale a >= 450 minutos totais jogados
Com esse corte, restam apenas jogadores com presença consistente — o ranking ganha credibilidade. Você pode ajustar para 3 ou 7 dependendo da quantidade de jogos do campeonato.
Campeonatos diferentes têm número de rodadas diferentes: Gaúcho tem menos jogos disponíveis que Paulista. Considere normalizar o corte por campeonato (ex: 40% das rodadas disputadas).

2b. Distribuição gols vs assistências
golsassistenciasgols_nao_penalti
Por quê: Separar gols de pênalti dos gols em jogo aberto mede melhor a capacidade técnica do jogador. O campo gols_nao_penalti já faz isso. Além disso, comparar gols vs assistências revela o perfil: finalizador (mais gols) ou criador (mais assistências).
Medidas DAX
Gols sem Penalti % = DIVIDE( SUM(fato_stats_jogadores_2025[gols_nao_penalti]), SUM(fato_stats_jogadores_2025[gols]) ) Perfil Ofensivo = DIVIDE( SUM(fato_stats_jogadores_2025[gols]), SUM(fato_stats_jogadores_2025[participacoes_gols]) ) -- próximo de 1 = finalizador / próximo de 0 = criador
No ranking final, destaque jogadores cujos gols vieram majoritariamente de jogo aberto. Penálti inflado pode distorcer o score.

Lógica do score (atacantes como exemplo)
-- Normalize cada métrica entre 0 e 1 antes de ponderar -- (valor - min) / (max - min) Score Atacante = 0.50 * [Part Gols 90 normalizado] + 0.30 * [Gols sem Penalti 90 normalizado] + 0.20 * [Min por Jogo normalizado]

Grupo 5

Análise de contexto — limitações a documentar

5a. Jogadores duplicados entre campeonatos
nomenome_campeonato
Por quê: Vitor Roque aparece com stats idênticas em Palmeiras e Mirassol no Paulista — isso indica duplicata no dado fonte (provavelmente jogador cedido com stats replicadas). Documentar essa limitação demonstra rigor analítico, que é um diferencial avaliado no teste.
Query de verificação
EVALUATE ADDCOLUMNS( SUMMARIZECOLUMNS( fato_stats_jogadores_2025[nome], fato_stats_jogadores_2025[nome_campeonato], "ocorrencias", COUNTROWS(fato_stats_jogadores_2025) ), "flag_dupla", IF([ocorrencias] > 1, "VERIFICAR", "OK") ) ORDER BY [ocorrencias] DESC
Inclua na seção de "Limitações" do seu entregável: jogadores emprestados podem ter stats replicadas na fonte. Na dúvida, deduplicar mantendo o registro com maior carga horária ou o do time principal.
